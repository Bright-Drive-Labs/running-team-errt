COMO USAR
================================================================================

1. Instalar dependencias:
   npm install fastify telegraf @supabase/supabase-js

2. Configurar .env:
   TELEGRAM_BOT_TOKEN=xxx
   SUPABASE_URL=xxx
   SUPABASE_ANON_KEY=xxx

3. Crear coach en BD:
   INSERT INTO athletes (...) VALUES (
     'coach-1', 'tenant-1', 'Coach', 'email@test.com', true, '123456789'
   );

4. Importar en servidor:
   import registerTelegramEndpoints from './handlers/fastifyTelegramEndpoints';
   await registerTelegramEndpoints(app);

5. Usar middleware en endpoints:
   import { requireTelegramAuth } from './middleware/validateTelegramUser';
   app.post('/api/test', { preHandler: requireTelegramAuth() }, handler);

6. Loguear acciones:
   import { logWorkoutPush } from './utils/auditLog';
   await logWorkoutPush({ ... });
